96#define _USE_LFN 0 /* 0 to 3 */
97#define _MAX_LFN 255 /* Maximum LFN length to handle (12 to 255) */
98/* The _USE_LFN option switches the LFN feature.
99/
100/ 0: Disable LFN feature. _MAX_LFN has no effect.
101/ 1: Enable LFN with static working buffer on the BSS. Always NOT reentrant.
102/ 2: Enable LFN with dynamic working buffer on the STACK.
103/ 3: Enable LFN with dynamic working buffer on the HEAP.
104/
105/ To enable LFN feature, Unicode handling functions ff_convert() and ff_wtoupper()
106/ function must be added to the project.
107/ The LFN working buffer occupies (_MAX_LFN + 1) * 2 bytes. When use stack for the
108/ working buffer, take care on stack overflow. When use heap memory for the working
109/ buffer, memory management functions, ff_memalloc() and ff_memfree(), must be added
110/ to the project. */
